# Workbooks Benefits Documentation

**Workbooks is a native desktop notebook environment that solves the pain points Jupyter ignores: secrets leak into notebooks, environments break constantly, and you're blind while cells run. Workbooks gives you encrypted secrets management, zero-setup Python environments, and real-time streaming output—all while keeping your notebooks standard .ipynb files.**

---

This directory contains benefit-by-feature documentation:

- **primary.md** - Core value propositions (what makes Workbooks worth using today)
- **secondary.md** - Supporting features that enhance the experience
- **other.md** - Non-obvious features that make Workbooks better than traditional tools
- **wishlist.md** - Compelling future features not currently planned
